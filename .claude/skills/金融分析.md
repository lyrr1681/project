# 金融趋势分析 Skill

分析金融市场趋势，生成每日结构化研报。

## 触发
用户提到「金融分析」「市场趋势」「每日研报」「金融趋势」时调用。

## 格式规范

### 结构：短期 → 中期 → 长期
每个时间维度下覆盖全部资产类别：A股、美股、港股、债券、黄金、科技、外汇、大宗商品、房地产。

### 写法
- 结论在前，论据在后
- 每段开头先给核心判断，再列数据支撑
- 无主观评分、无买卖建议
- 数据全部来自实际报道，搜索不到则标注「数据暂缺」

### 搜索步骤
1. 搜索 A股（沪指/深成指/创业板/科创50）当日行情
2. 搜索美股（道指/标普500/纳指）当日行情
3. 搜索港股（恒指/恒生科技）当日行情
4. 搜索美国10年/30年期国债收益率
5. 搜索黄金/白银现货价格
6. 搜索 WTI/布伦特原油价格
7. 搜索美元指数(DXY)/比特币/以太坊价格
8. 搜索重大宏观事件（央行决议/地缘冲突/政策变动）

### 输出
- 发送到对话（纯文本）
- 同时更新到 金融趋势分析.md
