# 金融趋势分析 Skill

分析金融市场趋势，生成每日结构化研报并持久化市场数据到本地数据库。

## 触发
用户提到「金融分析」「市场趋势」「每日研报」「金融趋势」时调用。

## 格式规范

### 结构：短期 → 中期 → 长期
每个时间维度下覆盖全部资产类别：A股、美股、港股、债券、黄金、科技、外汇、大宗商品、房地产。

### 写法
- 结论在前，论据在后
- 每段开头先给核心判断，再列数据支撑
- 无主观评分、无买卖建议
- 数据全部来自实际报道，搜索不到则标注「数据暂缺」

### 搜索步骤
1. 搜索 A股（沪指/深成指/创业板/科创50）当日行情
2. 搜索美股（道指/标普500/纳指）当日行情
3. 搜索港股（恒指/恒生科技）当日行情
4. 搜索美国10年/30年期国债收益率
5. 搜索黄金/白银现货价格
6. 搜索 WTI/布伦特原油价格
7. 搜索美元指数(DXY)/比特币/以太坊价格
8. 搜索重大宏观事件（央行决议/地缘冲突/政策变动）

## 存储层操作

### 0. 初始化（每次执行第一步）
```bash
sqlite3 /Users/bytedance/project/data/news.db "SELECT 1" 2>/dev/null || python3 /Users/bytedance/project/.claude/skills/init_db.py
```

### 1. 查询历史数据（开始搜索前）
在搜索当日行情前，先从 DB 拉取近期同类数据供对比分析：
```bash
# 查询最近 7 天各资产数据
sqlite3 /Users/bytedance/project/data/news.db \
  "SELECT asset_name, value, change_percent, data_date FROM market_data WHERE data_date > date('now', '-7 days') ORDER BY asset_name, data_date"
```
如果有返回数据，在分析中引用趋势变化。如果为空（首次运行），正常搜索即可。

### 2. 市场数据入库（完成搜索、写入文件前）
把搜索到的关键数据点逐条插入 market_data 表。数字型字段（股价、收益率、商品价格）：
```bash
python3 -c "
import sqlite3, sys
db = sqlite3.connect('/Users/bytedance/project/data/news.db')
db.execute('INSERT INTO market_data(asset_name, asset_category, value, change_percent, data_date, source) VALUES (?,?,?,?,?,?)',
    (sys.argv[1], sys.argv[2], float(sys.argv[3]), float(sys.argv[4]), sys.argv[5], sys.argv[6]))
db.commit()
db.close()
" "$asset_name" "$category" "$value" "$change_pct" "$(date +%Y-%m-%d)" "$source"
```

典型的数据点映射（每次搜索到后插入）：
| asset_name | category | value | change_pct |
|---|---|---|---|
| 沪指 | 股指 | 数值 | 涨跌幅 |
| 科创50 | 股指 | 数值 | 涨跌幅 |
| 道指期货 | 股指 | 数值 | 涨跌幅 |
| 标普500期货 | 股指 | 数值 | 涨跌幅 |
| 纳斯达克期货 | 股指 | 数值 | 涨跌幅 |
| 恒指 | 股指 | 数值 | 涨跌幅 |
| 恒生科技 | 股指 | 数值 | 涨跌幅 |
| 美债10年收益率 | 债券 | 数值 | 涨跌幅 |
| 美债30年收益率 | 债券 | 数值 | 涨跌幅 |
| 现货黄金 | 贵金属 | 数值 | 涨跌幅 |
| 美元指数 | 外汇 | 数值 | 涨跌幅 |
| WTI原油 | 商品 | 数值 | 涨跌幅 |

如果需要手动拼接带百分比的 change_percent，先用 python3 做数值提取。

### 3. 记录当日产出（写入文件后）
```bash
sqlite3 /Users/bytedance/project/data/news.db \
  "INSERT INTO daily_reports(report_date, report_type, file_path) VALUES (date('now'), '金融趋势', '/Users/bytedance/project/金融趋势分析.md')"
```

### 输出
- 发送到对话（纯文本）
