# 竞品调研 Skill
快速调研一个公司/产品/赛道的竞争格局。

## 触发
用户提到「调研」「竞品」「竞争对手」「行业分析」「市场格局」时调用。

## 搜索步骤
1. 搜索公司/产品基本信息（成立时间、团队、融资、规模）
2. 搜索市场份额与排名数据
3. 搜索核心产品/技术对比
4. 搜索近期重大动态（融资、收购、人事变动）
5. 搜索用户评价与口碑

## 输出格式
- **基本信息**：成立/团队/融资/规模
- **市场地位**：份额、排名、增速
- **核心竞争力**：技术/产品/渠道优势
- **近期动态**：重大事件时间线
- **风险与挑战**：竞争威胁、监管风险、技术瓶颈
