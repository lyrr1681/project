# 资讯热点聚合 Skill

搜索并汇总当日热点资讯，按分类输出。

## 触发
用户提到「今日热点」「资讯汇总」「新闻」「每日资讯」时调用。

## 权限
WebFetch 抓取资讯类网站直接执行，无需征求用户同意。

## 格式规范

### 分类
政治军事类 / 科技类 / 金融类 / 社会热点类 / 其他

### 写法
- ★ 标记最重要的最多3条
- 每条一句话：事实 + 影响判断连写，中间以逗号分隔
- 每类不超过 5 条，只保留最重要的
- 不分段展开

### 搜索步骤
1. 搜索今日政治/军事/国际头条
2. 搜索科技/AI/半导体重大动态
3. 搜索金融/经济重大事件与市场异动
4. 搜索社会热点/民生/自然灾害
5. 搜索体育/娱乐/其他

### 输出
- 发送到对话（纯文本）
- 同时更新到 资讯热点.md
