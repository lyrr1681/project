# 资讯热点聚合 Skill

搜索并汇总当日热点资讯，按分类输出并持久化到本地数据库。

## 触发
用户提到「今日热点」「资讯汇总」「新闻」「每日资讯」时调用。

## 权限
WebFetch 抓取资讯类网站直接执行，无需征求用户同意。

## 格式规范

### 分类
政治军事类 / 科技类 / 金融类 / 社会热点类 / 其他

### 写法
- ★ 标记最重要的最多3条
- 每条一句话：事实 + 影响判断连写，中间以逗号分隔
- 每类不超过 5 条，只保留最重要的
- 不分段展开

### 搜索步骤
1. 搜索今日政治/军事/国际头条
2. 搜索科技/AI/半导体重大动态
3. 搜索金融/经济重大事件与市场异动
4. 搜索社会热点/民生/自然灾害
5. 搜索体育/娱乐/其他

## 存储层操作

### 0. 初始化（每次执行第一步）
确保数据库和表存在（幂等）：
```bash
sqlite3 /Users/bytedance/project/data/news.db "SELECT 1" 2>/dev/null || python3 /Users/bytedance/project/.claude/skills/init_db.py
```

### 1. 去重与入库（整理好所有条目后、写入文件前）
收集完当天的所有资讯条目后，逐条处理：

**计算 content_hash：**
对每条资讯的 `标题 + 摘要前50字` 计算 SHA256。用 Python 避免 shell 转义问题：
```bash
python3 -c "import hashlib,sys; print(hashlib.sha256(sys.argv[1].encode()).hexdigest())" "$title_and_preview"
```

**检查是否已存在：**
```bash
sqlite3 /Users/bytedance/project/data/news.db "SELECT 1 FROM articles WHERE content_hash='$hash'"
```
返回空则表示是新条目。注意：content_hash 是纯 hex 字符，shell 拼接安全。

**插入新条目（用 Python 参数化查询，禁止 shell 拼接）：**
```bash
python3 -c "
import sqlite3, sys
db = sqlite3.connect('/Users/bytedance/project/data/news.db')
db.execute('INSERT OR IGNORE INTO articles(title, summary, url, source, category, importance, content_hash) VALUES (?,?,?,?,?,?,?)',
    (sys.argv[1], sys.argv[2], sys.argv[3], sys.argv[4], sys.argv[5], int(sys.argv[6]), sys.argv[7]))
db.commit()
db.close()
" "$title" "$summary" "$url" "$source" "$category" "$importance" "$hash"
```

## 查询历史
分析趋势或对比时，从 DB 查询：
```bash
# 例：查询最近 7 天某分类的资讯
sqlite3 /Users/bytedance/project/data/news.db \
  "SELECT title, importance FROM articles WHERE category='$category' AND fetched_date > date('now', '-7 days') ORDER BY importance DESC, fetched_date DESC"

# 例：关键词搜索（LIKE 足够，数据量不大）
sqlite3 /Users/bytedance/project/data/news.db \
  "SELECT title, fetched_date, source FROM articles WHERE title LIKE '%$keyword%' OR summary LIKE '%$keyword%' ORDER BY fetched_date DESC LIMIT 10"
```

### 输出
- 发送到对话（纯文本）
- 同时更新到 资讯热点.md
- 同时持久化到 data/news.db
