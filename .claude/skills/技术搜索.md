# 技术搜索 Skill
搜索技术文档、代码方案与最佳实践。

## 触发
用户提到「技术搜索」「查文档」「搜技术」「怎么实现」「代码搜索」时调用。

## 搜索来源
- GitHub Issues / Discussions
- StackOverflow
- 官方文档（MDN、Python.org、Go doc、Rust doc 等）
- 技术博客与公众号
- NPM/PyPI/CRAN 等包仓库

## 执行步骤
1. 确定技术栈与问题范围
2. 优先搜索官方文档
3. 搜索 GitHub Issues 中类似问题
4. 搜索 StackOverflow 高赞回答
5. 汇总方案对比，给出推荐方案与理由

## 输出
- 每个问题给出 2-3 个方案，标注优劣
- 优先推荐官方文档或社区验证方案
