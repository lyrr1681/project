# 学术搜索 Skill
搜索学术论文、研究报告与预印本。

## 触发
用户提到「论文」「研究」「学术」「Paper」「文献」「引用」时调用。

## 搜索来源
- Google Scholar
- arXiv / Papers With Code
- PubMed / medRxiv
- SSRN / RePEc
- CNKI（中国知网）

## 执行步骤
1. 确定研究领域与关键词
2. 搜索高被引论文
3. 搜索最新预印本
4. 搜索综述文章
5. 筛选权威来源

## 输出
- 论文标题 + 作者 + 年份 + 期刊/会议
- 核心发现一句话概括
- 被引次数（如有）
- 链接
- 最多列 10 篇，按影响力排序
